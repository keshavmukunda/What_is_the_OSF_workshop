This is a sample repo for the workshop [Organize your research project: an introduction to the Open Science Framework (OSF)](https://www.lib.sfu.ca/help/publish/research-data-management/organize-research-project-introduction-osf-workshop) offered by SFU Library, Simon Fraser University. 

During the hands-on workshop, participants will link this GitHub repo as an 'add-on' to our shared OSF project. The other files included in this repository are simply for demonstration.
